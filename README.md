# Emanuel Davidov | Software Engineer Portfolio
A professional, VS Code-themed portfolio website built with **Next.js**, **TypeScript**, and **CSS Modules**. 

## 🚀 About This Portfolio
This project is a personalized version of the VS Code-themed developer portfolio. It showcases my technical expertise in **Distributed Systems**, **Networking Protocols**, and **Infrastructure**, featuring projects built with **Java**, **Python**, and **Docker**[cite: 1].

## 🛠️ Tech Stack
- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** CSS Modules & Tailwind CSS
- **Icons:** React Icons (VS Code set)

## 📁 Project Structure
- `app/` - Next.js App Router pages (Home, About, Projects, Contact).
- `components/` - Visual Studio Code UI components (Explorer, Sidebar, Terminal, etc.).
- `data/` - Content data including project descriptions and technical skills[cite: 1].
- `styles/` - CSS modules for specific page layouts and themes.

## 💻 Featured Projects
- **The Digital Philologist**: Unsupervised NLP pipeline for authorship detection using BERT and Haar Wavelet Transforms[cite: 1].
- **Multi-threaded Chat System**: High-concurrency Java server with Docker integration and Prometheus/Grafana monitoring[cite: 1].
- **pandas-pyarrow**: Open-source contribution to the high-performance data processing ecosystem[cite: 1].

## 🛠️ Running Locally

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/emanueldavidov/portfolio_website.git](https://github.com/emanueldavidov/portfolio_website.git)
   npm install --legacy-peer-deps
   npm run dev
   ```

   Open http://localhost:3000 to view the result.

🎨 Themes
The portfolio supports multiple VS Code color themes (GitHub Dark, Dracula, Ayu, Nord, etc.), which can be toggled via the Command Palette (Ctrl + Shift + P) or the settings page.
